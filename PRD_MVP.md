# CareerOS MVP PRD (v0.1)

## Product Positioning
CareerOS helps users grow from task executor to strategic operator by turning daily work into a repeatable growth system.

## Primary Goals
- Convert daily work into reusable capability assets
- Improve manager communication quality
- Build growth feedback loop from execution data

## Target Users
- ICs with 1–5 years experience
- Analysts, PMs, engineers, project managers
- Users aiming to move from execution to strategic contribution

## Core Modules
- Dashboard/Today
- Projects
- Manager Update
- Insights
- People/Resource Map

## Success Metrics (MVP)
- Daily active usage on Dashboard
- Weekly manager update generated count
- Project records with milestone + next action
- Insight records converted to recommendation

## AI MVP
- Notes -> Manager Update draft
- Phenomenon -> Insight draft (next sprint)

## Delivery Plan
- Phase 1: static pages
- Phase 2: local/mock data
- Phase 3: Flask + SQLite APIs
- Phase 4: AI endpoints
